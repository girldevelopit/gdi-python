What is Programming?

Giving a computer instructions to do a task.

A program is one or more files of code, where each part solves a specific task.

"Under the hood" the computer reads the program as binary code.

What is Python?

Just like with human languages (English, Spanish, etc.) there are many programming langauges (Python, HTML, C++,etc.)

Python is a general purose programming language

Why Python?

Many applications

Easy to read and use

Popular language (many resources available)

Good learning language (functional programming, object-oriented)

When can you use Python?

Web Applications

Variables

					
						#this is a string
						print(type("Hello World"))

						#this is a integer (no decimals)
						print(type(3))

						#this is a float (decimal places)
						print(type(3.1))

						#you can create functions to do a specific task

						def add(x, y)
							answer = x + y
							print(answer)

						add(2, 3)

Boolean Data Type

					
						#boolean data types are True/False
						a = True
						print(type(a))

						b = False
						print(type(b))

Boolean Expressions

					
						#boolean expressions evaluate to True or False
						a = 1
						b = 2


						print(a > b) #is a greater than b?
						print(a < b) #is a less than b?
						print(a == b) #is a equal to b? 
						print(a != b) #is a not equal to b?

If, Else statements

					
						#you can have your code make choices with if, else
						a = 1
						b = 2

						if a == b: #if the statement evaluates to True
						  print("a is equal to b")
						else: #if the statement evaluates to False
						  print("a is not equal to b")

If, Else statements cont.

					
						#if, else statements with strings
						day = "Friday"

						if day == "Friday":
						  print("Happy Friday!")
						else: 
						  print("Have a good day!")

Putting it all together!

					
						def hello(day):
						  print("Happy "+ day)
						  if day == "Friday":
						    print("It's almost the weekend!")
						  else:
						    print("Have a good day!")

						hello("Saturday")
						hello("Friday")

Lists

					
						
						#list can hold multiple elements of different types
						my_list = ["hello", 1, 3.5]
						
						#lists start at 0
						my_list[0]
						
						#how many elements are in my list?
						print(len(my_list))

Using "elif" (else+if) to add another expression

					
						def hello(day):
						  print("Happy "+ day)
						  if day == "Friday":
						    print("It's almost the weekend!")
						  elif day in ["Saturday", "Sunday"]:
						    print("Woohoo! It's the weekend!")  
						  else:
						    print("Have a good day!")

						hello("Saturday")
						hello("Friday")
						hello("Tuesday")

Making the code more robust

					
						def hello(day):
							print("Happy "+day+"!")
							#change day into all lowercase
							day = day.lower()
							if day == "friday":
								print("It's almost the weekend!")
							elif day in ["saturday", "sunday"]:
								print("Woohoo! It's the weekend!")
							else:
								print("Have a good day")

						hello("saturday")
						hello("FRIDAY")
