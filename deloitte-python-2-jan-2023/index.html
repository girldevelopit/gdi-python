Overview

  • File input/output
  • List comprehension
  • Argument parsing
  • Error handling
  • Making HTTP requests/parsing JSON responses

Coding environment setup

We're using Replit, a web-based coding environment, for this class. To get started, create a Replit account.

Replit tips:

  • To use the class exercises below, open the link and click the "Fork repl" buttton to copy it to your own account
  • Each exercise contains 2 files: main.py and solution.py. Work on the exercise in the main.py file, as the Replit Run button only runs code in main.py
  • You can run code by typing directly into the Console tab or by typing into the main.py file and clicking the Run button.
