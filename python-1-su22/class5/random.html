Random numbers

Random numbers in Python

The random module contains various functions for generating randomness.


    import random

    rando = random.randint(1, 5)   # 1, 2, 3, 4, or 5

    randy = random.randrange(1, 6) # 1, 2, 3, 4, or 5

Random example


    def random_rgb():
        r = random.randint(0, 255)
        g = random.randint(0, 255)
        b = random.randint(0, 255)
        return r, g, b

    r1,g1,b1 = random_rgb()

Can computers be truly random?

No. Computers use pseudo-random number generators, which are considered good enough for most purposes.

For true randomness (if it exists), you have to find random phenomena in the world, like atmospheric noise.
